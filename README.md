# Excel-financial-analysis-Project
Project Overview:
This project demonstrates beginner-level financial data analysis using Microsoft Excel. The workbook includes structured financial calculations such as depreciation and net income estimation using Excel Formulas. 
The objective of this project was to understand how financial variables interact and how excel can be used as a tool for organizing and analyzing financial data efficiently. 

Skills used:
1) Microsoft Excel
2) Basic financial analysis
3) Excel formulas and functions
4) Data structuring in spreadsheets
5) Logical calculation workflow

   Project Contents:
   This Excel workbook includes:
   1) Depreciation calculation sheet
   2) Net Income estimation sheet
   3) Structured financial data tables
   4) Formula-based financial computations
  
      DATASET information:
      The dataset used in this project was created for practice purposes to simulate financial statement analysis and strengthen Excel calculation skills.

      Key Learnings:
      Through this project, I learned:
      1) How depreciation affects financial statements
      2) How to calculate net income using Excel formulas
      3) How to structure financial datasets clearly
      4) How to organize analytical workflows inside spreadsheets.

         Tools used:
         Microsoft Excel
